### 2. Assigning Member Permissions

Once the mailbox is created, you must authorize specific users to access it. There are two primary permission types:

- Read and Manage (Full Access): Allows the user to open the mailbox and view its folders.

- Send As / Send on Behalf: Allows the user to reply to messages using the shared email address.
